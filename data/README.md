# Data Directory

## Dataset: Craigslist Vehicles

**Source:** [Used Cars Dataset — Kaggle](https://www.kaggle.com/datasets/austinreese/craigslist-carstrucks-data)

## Setup Instructions

1. Go to the Kaggle link above and download the dataset.
2. Extract the archive — you will get a file called `vehicles.csv`.
3. Place `vehicles.csv` directly inside this `data/` folder.

The final path should be:

```
data/vehicles.csv
```

> **Note:** This file is ~1.4 GB and is excluded from version control via `.gitignore`. Each team member must download it locally.
